# 创建套餐(通过api创建套餐后，需要联系管理员审核通过之后方可使用)

1. 请求地址:</br>
   https://{domain}/payment/mandate/plan/add


2. 请求方式:</br>
   POST


3. Content-Type:</br>
   application/json; charset=utf-8


4. 参数说明</br>

|参数名|必选|类型| 说明                                                              |
|--|--|--|-----------------------------------------------------------------|
|appId|是|string| 接入方应用id                                                         |
|planName|是|string| 套餐名称                                                            |
|recurringAmount|是|string| 套餐单次扣款金额                                                        |
|recurringTimes|是|string| 套餐总扣款次数                                                         |
|billingCycle|是|string| 扣款周期 0-daily,1-weekly,2-monthly,3-yearly,5-quarterly,6-twoWeeks |
|signature|是|string| 签名，验签算法请参考```安全说明```文档                                          ||


5. 请求参数示例

```json
{
   "appId": "0000000253",
   "planName": "planName1210",
   "recurringAmount": "10",
   "recurringTimes": "60",
   "billingCycle": "2",
   "signature": "898269e617996557924e2dffb576d558"
}
```


6. 响应数据示例
``` json
{
  "code": 200,
  "message": "success",
  "data": 1
}
```
