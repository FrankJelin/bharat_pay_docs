# 获取可用的订阅计划(套餐)列表

1. 请求地址:</br>
   https://{domain}/payment/mandate/plan/list


2. 请求方式:</br>
   POST


3. Content-Type:</br>
   application/json; charset=utf-8


4. 参数说明</br>

|参数名|必选|类型| 说明                     |
|--|--|--|------------------------|
|appId|是|string| 接入方应用id                |
|signature|是|string| 签名，验签算法请参考```安全说明```文档 ||


5. 请求参数示例

```json
{
   "appId": "0000000253",
   "signature": "ec038a938cc5cd1118690dc56981f579"
}
```
